# MajorStoryPPT

“我的专业，我的国”主题交互式演示网页项目。  
主文件为单页集成版，包含四部分叙事与统一结语，支持分段主题、页面动效、交互选择与结束黑屏。

## 项目结构

- `major-country-data-science.html`：主演示文件（推荐直接打开）
- `image_for_groupmajorPPT/`：主文件使用的图片资源
- `slr/`：第三部分（武汉）相关素材与原稿
- `fzc/`：第四部分（华科群星）相关素材与原稿
- `gaigekaifang (1).html`：第二部分原稿（改革开放）

## 使用方式

1. 克隆仓库到本地
2. 直接用浏览器打开 `major-country-data-science.html`
3. 建议使用全屏演示，并在更新后按 `Ctrl + F5` 强制刷新缓存

## 主要交互说明

- 首页进入第一部分（新中国）
- 第一部分完成后衔接第二部分（改革开放）
- 第二部分后进入第三部分（武汉）
- 第三部分后进入第四部分（华科群星）
- 最后进入统一结语页，可点击“🏁 结束演讲 · 谢谢聆听”触发黑屏结束页
